### Servidor Web + Express + HBS + Heroku
- Usaremos express 
- Contenido estatico
- Handlebars => helpers, parciales, variables 
- Hacer publicas algunas carpteas
- HTTP es un paquete que nos permite crear un web server
- HTTP no se instala
Recuerda instalar las dependencias:

```
npm install
```

Para ejecutar:

```
npm start
```
